# nakama

This repo created for applying sdet on toped.

## how to run the test
install npm 
```bash
npm install
```
- using npm 
```bash
npx cypress open
```
##

install yarn 
```bash
yarn add
```
- using yarn 
```bash
yarn run cypress open

```
